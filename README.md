**Mesh** treats context as a *managed budget*, not an unbounded side effect of the agent loop. Repository state is materialized locally into a field-weighted lexical index, dense code-embedding vectors, and a deterministic **Repository Intelligence Graph (RIG)** — then assembled into budgeted, cited evidence **before** each model turn, instead of accumulating through exploratory tool calls.

### Why it matters

- **🎯 Better localization, fewer tokens** — Hit@1 **79.9%** on a 229-task suite across six external repositories, at **−69.7%** tokens per query versus a competent grep-based agent.
- **🕸️ Graph-native** — symbol-aware exact lookup and dependency-graph impact, where pure embeddings and grep structurally fall short.
- **💸 Cost-normalized** — every quality metric is reported alongside its token and dollar cost.
